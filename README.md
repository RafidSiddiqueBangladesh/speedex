# Video Speed Controller Chrome Extension

A Chrome extension that lets you speed up YouTube and any video playing in Chrome up to 4x speed with just a click!

## Features

- **Quick Speed Controls**: Preset buttons for 0.5x, 0.75x, 1x, 1.25x, 1.5x, 1.75x, 2x, 3x, and 4x speeds
- **Custom Speed Slider**: Fine-tune any speed between 0.25x and 4x
- **Works Everywhere**: YouTube, Vimeo, and any HTML5 video player
- **Easy Reset**: One-click reset back to normal speed
- **Keyboard Shortcuts**: Ctrl+Shift+= to speed up, Ctrl+Shift+- to slow down

## Installation

### Option 1: Manual Installation (Developer Mode)

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Navigate to the folder containing this extension and select it
5. The extension will now appear in your Chrome extensions

### Option 2: Package as CRX (Advanced)

1. Go to `chrome://extensions/`
2. Enable Developer mode
3. Right-click on the extension and select "Package extension"

## Keyboard Shortcuts

- **Ctrl + Shift + =** : Speed up by 0.25x
- **Ctrl + Shift + -** : Slow down by 0.25x

## Supported Websites

- YouTube
- Vimeo
- Netflix (some versions)
- Twitch
- Any website with HTML5 video players

## Notes

- Speed changes apply to the active tab only
- Each tab can have a different playback speed
- The extension works best with HTML5 video players

## Troubleshooting

**Extension not working on a video:**
- Make sure the website uses HTML5 video player
- Try refreshing the page
- Some videos in iframes might not be controllable due to cross-origin restrictions

**Keyboard shortcuts not working:**
- Make sure you're focused on the webpage (not another app)
- Try using the popup buttons instead
