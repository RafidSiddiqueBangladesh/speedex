Note: This tool helps you understand how to get your extension ID.

Your extension ID will be generated automatically when you first load it in Chrome Developer Mode.

Method 1: Automatic (Recommended)

  1. Go to chrome://extensions/
  2. Enable "Developer mode"
  3. Click "Load unpacked" and select your extension folder
  4. Your extension ID will be displayed (16 characters)
  5. Copy this ID - you'll need it for the update manifest

Method 2: Manual CRX Signing (Advanced)

For auto-updates, you can sign your extension:

Windows PowerShell Commands:

# 1. Generate private key
# Download Chrome as CRX generator or use command line tools
# Your extension ID will be shown when you pack it

  1. Load your extension in Chrome (chrome://extensions/)
  2. Write down the Extension ID shown (16 characters)
